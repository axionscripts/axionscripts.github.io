Configuration Examples

Basic Configuration

Config = {}

-- Framework Configuration
Config.Framework = "esx" -- "esx", "qb-core", or "standalone"

-- Notification Settings
Config.Position = "top-right" -- "top-left", "top-right", "bottom-left", "bottom-right"
Config.MaxNotifications = 5
Config.DefaultDuration = 5000
Config.SoundEnabled = true
Config.AnimationDuration = 300

Framework Integration

-- ESX Integration
if Config.Framework == "esx" then
    ESX = exports["es_extended"]:getSharedObject()
    
    -- Override ESX notifications
    function ESX.ShowNotification(message, type, duration)
        exports['axion-notify']:notify(message, type, duration)
    end
end

-- QBCore Integration
if Config.Framework == "qb-core" then
    QBCore = exports['qb-core']:GetCoreObject()
    
    -- Override QBCore notifications
    function QBCore.Functions.Notify(message, type, duration)
        exports['axion-notify']:notify(message, type, duration)
    end
end

Configuration Tips

Custom Colors

Use hex color codes for custom notification colors. Ensure good contrast with white text for readability.

Sound Configuration

Add custom sound files to the resource and reference them in the configuration. Use .ogg or .wav format for best compatibility.

Position Settings

Choose notification position based on your UI layout. "top-right" is most common, but "bottom-right" works well for mobile-style interfaces.
