Axion Notify - Usage Guide

Complete usage guide and API documentation for Axion Notify

Client-Side Usage

Basic Notification

-- Simple notification
exports['axion-notify']:notify('Hello World!', 'success', 5000)

-- With different types
exports['axion-notify']:notify('Something went wrong!', 'error', 3000)
exports['axion-notify']:notify('Warning message', 'warning', 4000)
exports['axion-notify']:notify('Information', 'info', 5000)

Advanced Notification

-- Advanced notification with custom title and icon
exports['axion-notify']:notifyAdvanced(
    'Custom Title',           -- Title
    'This is a custom message', -- Message
    'info',                   -- Type
    5000,                     -- Duration
    'fas fa-star'             -- Custom icon
)

Custom Notification

-- Fully custom notification
exports['axion-notify']:notifyCustom(
    'Custom Title',
    'Custom message with custom styling',
    5000,
    'fas fa-heart',
    '#ff6b6b'  -- Custom color
)

Server-Side Usage

Notify Specific Player

-- Basic server notification
exports['axion-notify']:notifyPlayer(source, 'Welcome to the server!', 'info', 5000)

-- Advanced server notification
exports['axion-notify']:notifyPlayerAdvanced(source, {
    title = 'Server Message',
    message = 'This is a server notification',
    type = 'success',
    duration = 5000,
    icon = 'fas fa-server'
})

Notify All Players

-- Notify all players
exports['axion-notify']:notifyAll('Server restart in 5 minutes', 'warning', 10000)

-- Advanced notification for all players
exports['axion-notify']:notifyAllAdvanced({
    title = 'Announcement',
    message = 'Important server announcement',
    type = 'info',
    duration = 8000,
    icon = 'fas fa-bullhorn'
})

Framework Integration

ESX Integration

-- ESX notifications will automatically use Axion Notify
ESX.ShowNotification('This will use Axion Notify!', 'success', 5000)

QBCore Integration

-- QBCore notifications will automatically use Axion Notify
QBCore.Functions.Notify('This will use Axion Notify!', 'success', 5000)

Test Commands

/testnotify error [message] Client

Test an error notification

/notifyplayer [playerid] [message] Server

Send a notification to a specific player

/notifyall [message] Server

Send a notification to all players
