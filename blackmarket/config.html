Timing Configuration

Config.Timing = {
    mode = "schedule", -- "duration", "schedule", or "always"
    duration_minutes = 60, -- For duration mode
    schedule = {
        start_hour = 21, -- 9 PM (24-hour format)
        end_hour = 1,    -- 1 AM (24-hour format) - supports midnight crossing
        days = {1,2,3,4,5,6,7} -- Days of week (1=Monday)
    },
    always = {
        respawn_interval = 300 -- Respawn interval in seconds when shop despawns
    }
}

Configuration Tips

Timing Modes

Use "schedule" mode for realistic black market hours, "duration" for time-limited events, or "always" for permanent shops with respawn intervals.

Payment Types

Configure different payment types per shop type. Use "black_money" for illegal items, "clean_money" for legal items, or "crypto" for modern payments.

Stock Management

Use "random" stock type for dynamic inventory, "fixed" for consistent stock, or "unlimited" for always available items.
