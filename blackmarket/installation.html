Axion Black Market - Installation

Complete installation guide for the Axion Black Market script

Required for NPC interactions in Axion Black Market

Download

ox_lib

Required for UI components and notifications

Download

ESX or QBCore

Player framework (choose one)

ESX QBCore

ox_inventory

Optional but recommended for inventory management

Download

Installation Steps

1

Purchase and Download

Purchase the script from our Tebex store and download the files:

Get Axion Black Market

Purchase the script to get access to download links and support

Purchase Now
2

Configure Dependencies

Add the required resources to your server.cfg:

# Required dependencies
ensure ox_target
ensure ox_lib
ensure es_extended  # or qb-core

# Axion Black Market
ensure axion-blackmarket
3

Configure the Script

Edit config.lua to match your server setup:

-- Framework Configuration
Config.Framework = "esx" -- Change to "qb-core" if using QBCore

-- Shop Locations (add your preferred locations)
Config.ShopLocations = {
    {
        coords = vector3(-1037.0, -2737.0, 20.2),
        heading = 240.0,
        name = "Airport Terminal",
        blip = {
            sprite = 52,
            color = 1,
            scale = 0.8,
            name = "Black Market"
        }
    },
    -- Add more locations...
}
4

Customize Items

Edit data/shop_items.lua to add your own items:

ShopItems.weapon_items = {
    {
        name = "weapon_pistol",
        label = "Pistol",
        price = 2500,
        category = "weapons",
        weight = 1.0,
        description = "A standard pistol for self-defense"
    },
    -- Add more items...
}
5

Start the Resource

Restart your server or start the resource manually:

-- To manually start, type the following in the server console (or f8)
start axion-blackmarket

Troubleshooting

Script not starting

  • Check if all dependencies are installed
  • Verify the resource name in server.cfg
  • Check console for error messages

UI not appearing

  • Ensure ox_target is working
  • Check if ox_lib is properly configured
  • Verify NUI files are in correct location

Payment issues

  • Check framework configuration (ESX/QBCore)
  • Verify payment method settings
  • Ensure player has sufficient funds

Items not showing

  • Verify item names match your inventory system
  • Check if items exist in your database
  • Ensure proper framework configuration
