Axion Black Market - Usage Guide

Complete usage guide and API documentation for Axion Black Market

For Players

Find a Shop

Look for black market blips on the map or NPCs at configured locations. Shops spawn randomly based on your timing configuration.

Interact with NPC

Use ox-target to interact with the shop NPC. The interaction will open the shop interface.

Browse Items

Use the category filters to find items. Only categories with available items will be shown.

Purchase Items

Select an item, choose quantity, and payment method. Items will be added to your inventory upon successful purchase.

Admin Commands

/blackmarket spawn Admin

Spawns a black market shop at a random location

/blackmarket despawn Admin

Despawns all active black market shops

/blackmarket reload Admin

Reloads the black market configuration

/blackmarket status Admin

Shows the current status of all shops

Configuration Examples

Shop Types Configuration

Config.ShopTypes = {
    ["weapons"] = {
        name = "Weapon Dealer",
        model = "s_m_y_dealer_01",
        items = "weapon_items",
        payment_type = "black_money",
        price_multiplier = 1.0,
        randomized = false,
        max_items = 10
    },
    ["drugs"] = {
        name = "Drug Dealer",
        model = "s_m_y_dealer_01",
        items = "drug_items",
        payment_type = "clean_money",
        price_multiplier = 1.2,
        randomized = true,
        max_items = 15
    }
}

Timing Configuration

Config.Timing = {
    mode = "schedule", -- "duration", "schedule", or "always"
    duration_minutes = 60, -- For duration mode
    schedule = {
        start_hour = 21, -- 9 PM (24-hour format)
        end_hour = 1,    -- 1 AM (24-hour format) - supports midnight crossing
        days = {1,2,3,4,5,6,7} -- Days of week (1=Monday)
    },
    always = {
        respawn_interval = 300 -- Respawn interval in seconds when shop despawns
    }
}

Item Configuration

ShopItems.weapon_items = {
    {
        name = "weapon_pistol",
        label = "Pistol",
        price = 2500,
        category = "weapons",
        weight = 1.0,
        description = "A standard pistol for self-defense",
        stock = {
            type = "random", -- "unlimited", "fixed", or "random"
            amount = 5, -- For fixed type
            range = {min = 2, max = 8} -- For random type
        }
    }
}
